## Run authorization-form locally

### Step 1: Clone project

```sh
git clone https://github.com/MikhailEzhov/authorization-form.git authorization-form
cd authorization-form
```

### Step 2: Install npm dependencies

```sh
npm install
```

### Step 3: Usage

```sh
npm start
```
